# gravity-simulator
A 3D gravity simulator using OpenGL

## Download and Run
Clone the repository (`git clone git@github.com:gequalspisquared/gravity-simulator.git`), 
cd into `gravity-simulator`, and enter `make run`.

## Dependencies
OpenGL
GLAD (v4.6.0 Core or greater)
GLFW
GLM
Assimp
